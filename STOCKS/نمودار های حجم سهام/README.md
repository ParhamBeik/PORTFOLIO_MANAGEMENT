# نمودار های حجم سهام (Stock Volume Charts)

This folder contains the script to generate candlestick + net-flow charts (individual or institutional) from candlestick and historic legal/real data.

## Configuration

Edit the top of `generate_volume_price_charts.py`:

- **MODE**: `"individual"` (حقیقی) or `"institutional"` (حقوقی). Chooses which net flow series to plot (Buy_I_Value - Sell_I_Value vs Buy_N_Value - Sell_N_Value).
- **TIMEFRAME**: `"daily"`, `"weekly"`, or `"monthly"`. Data is always read as daily; weekly and monthly aggregate OHLC and flow before plotting.

## Output

Charts are written under this folder:

- `{TIMEFRAME}/{MODE}/{industry}/{symbol}.png`

Example: `daily/individual/سيمان آهك و گچ/سشمالح.png`

Running with different MODE or TIMEFRAME writes to different subfolders so outputs do not overwrite each other.

## Data sources

- Candlesticks: `../CANDLESTICKS/data/FETCH_CANDLESTICK_DATA/*/*/Historical/adjusted.json`
- Flow: `../HISTORIC DATA/data/FETCH_HISTORIC_DATA/{industry}/{symbol}/دیتای حقیقی و حقوقی.json`

Candlestick-only charts (no flow) are generated by `generate_adjusted_log_candlestick_charts.py` in `CANDLESTICKS/scripts/`.
